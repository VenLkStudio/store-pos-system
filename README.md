# 🏪 Розничный магазин — WinForms C# приложение

## Описание
Система управления розничным магазином на C# WinForms с подключением к SQL Server.

## Требования
- .NET Framework 4.8 или .NET 6+ (WinForms)
- Microsoft SQL Server 2017+ или SQL Server Express
- Visual Studio 2019/2022

---

## Установка и запуск

### Шаг 1: Создание базы данных

1. Откройте **SQL Server Management Studio (SSMS)**
2. Подключитесь к серверу
3. Откройте файл `Database/CreateDatabase.sql`
4. Нажмите **Execute (F5)**

База данных `RetailShop` будет создана с тестовыми данными.

### Шаг 2: Настройка проекта

1. Откройте `RetailShop.sln` в Visual Studio
2. Если нужно, измените строку подключения в `Database/DbHelper.cs`:
   ```csharp
   private static string _connectionString = 
       "Server=localhost\\SQLEXPRESS;Database=RetailShop;Integrated Security=True;";
   ```

### Шаг 3: Сборка и запуск

```bash
dotnet build
dotnet run
```

Или нажмите **F5** в Visual Studio.

## Функциональность по ролям

### 🟦 Оператор
- Приём партий товара
- Сверка с документами (накладная, счёт-фактура, сертификат качества)
- Отправка товара на склад
- Просмотр товаров и остатков

### 🟨 Администратор
- Все функции оператора
- Расчёт розничных цен (с наценкой)
- Формирование заявок в торговый зал

### 🟥 Товаровед
- Просмотр товаров и остатков склада
- Оценка несоответствия товара
- Оформление возвратных накладных поставщику

### 🟩 Старший кассир
- Открытие/закрытие смен на кассах
- Реализация товаров (оформление чеков)
- Формирование кассовых отчётов
- Инвентаризация (сверка фактических остатков с базой)
