# Розничный магазин — WinForms C#

## Быстрый старт

### 1. Создать базу данных
Откройте **SSMS**, подключитесь к SQL Server,
откройте файл `Database/CreateDatabase.sql` и нажмите **F5**.

### 2. Открыть проект
Откройте `RetailShop.sln` в **Visual Studio 2019/2022**.

### 3. Проверить строку подключения
В файле `Database/DB.cs` при необходимости измените:
```csharp
public static string ConnectionString =
    "Server=localhost\\SQLEXPRESS;Database=RetailShop;Integrated Security=True;";
```

### 4. Запустить (F5)
При первом запуске появится диалог настройки подключения.
